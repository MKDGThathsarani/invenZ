## 📖 About / විස්තරය

**InvenZ** is a modern, feature-rich inventory management system built specifically for Sri Lankan businesses. It helps you track products, manage suppliers, handle orders, and monitor stock levels with real-time alerts.

**InvenZ** යනු ශ්‍රී ලාංකික ව්‍යාපාර සඳහා විශේෂයෙන් නිර්මාණය කරන ලද නවීන ඉන්වෙන්ටරි කළමනාකරණ පද්ධතියකි. මෙය ඔබට භාණ්ඩ ලුහුබැඳීම, සැපයුම්කරුවන් කළමනාකරණය, ඇණවුම් හැසිරවීම සහ තථ්‍ය කාලීන අනතුරු ඇඟවීම් සමඟ ගබඩා මට්ටම් නිරීක්ෂණය කිරීමට උපකාරී වේ.
